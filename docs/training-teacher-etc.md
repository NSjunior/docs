- เมนูแสดง User ผู้ใช้งานอยู่มุมด้านบน ขวามือ เมื่อคลิกเข้าไปแล้ว User ใช้งานจะสามารถดูข้อมูลของตัวเองได้ สามารถดาวน์โหลดคู่มือการใช้งานได้และเมื่อต้องการออกจากระบบก็สามารถเลือกที่เมนูนี้เช่นกัน
